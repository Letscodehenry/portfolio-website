Works & Projects

Hotel Reservation Sytem

Hotel reservation system built with Tkinter in Python facilitates booking accommodations by capturing essential details such as booking number, contact information, payment method, check-in and check-out dates, room preferences, and additional amenities. Users can easily save, update, search, and delete reservations through intuitive graphical user interfaces. The system stores reservation data in an Excel file, providing a structured and accessible format for managing bookings. With features like search functionality and data validation, it ensures efficiency and accuracy in handling reservations for hotels or similar establishments.

League of Legends Heroes Management System

A text-based hero management system using JSON operates by storing hero data in a JSON file. The system utilizes Python's `json` module to read and write data to this file, enabling functionalities such as adding new heroes, updating existing hero attributes, and deleting heroes. Users interact with the system through a command-line interface, inputting commands to perform various operations on the hero data stored in the JSON file.
